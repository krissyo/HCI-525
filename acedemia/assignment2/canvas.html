HTML Elements - Canvas

A very cool entry into the JS1K challenge, shame your browser doesn't support it.
This is my ISC 325 assignment 2. I am enjoying experimenting with html5. This page is dedicated to the canvas functionality. The canvas html5 feature is for displaying 2D shapes and bitmap images. The canvas can be controlled through javascript calls and is essentially like an art canvas. You must create and/or insert content through a script tag at the bottom of the body - no code being rendered can be written inside the canvas tags. Within the canvas tag it is best to make a note of the content being displayed incase a user is not using a browser which supports html5. Width and height are needed to specify the dimensions. The canvas element is my favorite part of html5 as I feel it is innovative as it compeltely replaces flash and embedded objects. It also allows mobile broswers to display the same content as desktop broswers.
